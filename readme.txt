Пояснение по пунктам лабораторной работы №9: 
Анимация:
menuAnimation:(во всех страницах, меню)
Эта анимация позволяет меню подниматься и опускаться при наведении курсора.

submenuAnimation:(во всех страницах, подменю)
Эта анимация вызывает движение подменю вверх и вниз.

shakeButton:(страница "Уроки", кнопки)
Создает эффект тряски кнопки.

button:(страница Уроки/Задание книпки задании, страница Языкавой тест/Казахский(Русский, Английский) кнопка Проверить результат, Войти/Зарегестрироваться соответствующие названия)
Применяет тень с радиусом размытия и красным цветом при наведении на кнопку. 

Трансформации:
language-buttons (страница Языкавой тест/кнопка "Казахский")
При наведении увеличение размера кнопки и небольшой поворот по часовой стрелке. Плавное изменение цвета фона и размера кнопки.

language-buttons (страница Языкавой тест/Блок "Русский")
При наведении увеличение размера кнопки и делает наклон кнопки по горизонтали. Плавное изменение цвета фона и размера кнопки.

language-buttons (страница Языкавой тест/Блок "Английский")
При наведении увеличение размера кнопки и делает наклон кнопки по вертикали. Плавное изменение цвета фона и размера кнопки.
